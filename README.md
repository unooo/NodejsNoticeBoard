# NodejsNoticeBoard


Javascript Async/Await 과 Ajax, NodeJs를 활용한 Restful API NoticeBoard 토이 프로젝트입니다.

1.이미지 업로드용 게시판 및 커뮤니티 서비스 어플리케이션 
 
2.해시태그, 쪽지, 카테고리별 검색을 통한 REST API 집중 활용 연습 
 
3.AJAX , Async/Await 을 최대한 중점적으로 이용한 어플리케이션
